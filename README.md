# Image Gallery

## Description
A fully responsive web-based Image Gallery built using HTML, CSS, and JavaScript. Users can explore images, filter by category, like/favorite images, download them, and toggle between light and dark modes for a personalized experience.

## Features
- Responsive gallery that works seamlessly on desktop, tablet, and mobile.
- Search functionality to quickly find images.
- Category filters for easy navigation through different image types.
- Like/Favorite button to save preferred images.
- Download feature to save images locally.
- Dark mode toggle for user-friendly viewing in low-light conditions.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Tools: VS Code, Git & GitHub
